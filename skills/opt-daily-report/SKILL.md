# OPT 每日报告技能

## 功能说明

此技能帮助 CoS Agent 自动生成每日简报和周报，让老板用最短的时间了解公司全貌。

## 每日简报生成流程

每天 08:00，CoS 执行以下步骤：

1. 读取各 Agent 昨日的 `memory/YYYY-MM-DD.md` 文件。
2. 读取 `workspace/projects/` 目录，获取各项目最新状态。
3. 读取 `knowledge-base/lessons-learned/` 获取最新踩坑记录。
4. 汇总生成简报，发送到 `#hq` 频道。

## 每日简报模板

```markdown
# 📊 OPT 每日简报 — {YYYY-MM-DD}

## 🔍 灵感嗅探
{今日值得关注的行业动态、设计趋势或商业机会，2-3条，每条附上信息来源}

## ⚙️ 生产感知
| 项目 | 状态 | 负责人 | 预计完成 | 备注 |
|---|---|---|---|---|
| {项目名} | 🟢进行中 / 🟡待处理 / 🔴阻塞 | {Agent} | {日期} | {简要说明} |

## 📦 资产沉淀
{昨日新增的知识条目、技能文件或可复用模板，列出名称和简要说明}

## ⚠️ 需要您关注
{需要人类老板决策或确认的事项，若无则写"暂无"}

---
*本报告由 CoS Agent 自动生成 · {生成时间}*
```

## 周报模板

每周一 09:00 生成：

```markdown
# 📈 OPT 周报 — {YYYY} 第 {WW} 周

## 本周成果
{本周完成的主要任务和交付物}

## 关键指标
- API 消耗：{Token 数} / 预算 {预算}
- 完成任务：{A类} 个 Action，{P类} 个 Project 里程碑
- 新增知识：{条数} 条知识条目，{个数} 个新技能

## 下周计划
{下周的主要工作重点}

## 系统健康
{来自 Ops 的系统状态摘要}
```
