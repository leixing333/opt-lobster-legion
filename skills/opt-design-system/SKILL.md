# 技能：OPT 品牌设计系统

## 简介
本技能为 OPT 一人设计公司提供**统一的品牌设计规范**，确保所有输出物（网站、报告、演示文稿、社交媒体内容）在视觉上保持一致性。它是 Builder 和 CoS 在创建任何视觉内容时的必读规范。

## 适用角色
**Builder**（主要）、**CoS**（参考）

## 触发条件
- 在创建任何视觉内容（网站、报告、演示文稿）前必须读取本技能。
- 在选择颜色、字体、布局时必须参考本技能。
- 在审查视觉内容时必须对照本技能进行检查。

## 设计规范

### 色彩系统（Color System）

| 色彩角色 | 颜色名称 | HEX 值 | 使用场景 |
|---|---|---|---|
| **主色** | 深蓝黑 | `#0A0E1A` | 背景、主要区域 |
| **强调色 1** | 龙虾红 | `#FF3B3B` | 主要 CTA、重要标签 |
| **强调色 2** | 科技青 | `#00D4FF` | 链接、次要强调 |
| **强调色 3** | 紫色光 | `#7B61FF` | 渐变、装饰元素 |
| **文字主色** | 纯白 | `#FFFFFF` | 主要文字 |
| **文字次色** | 灰白 | `#A0A8B8` | 次要文字、说明文字 |

### 字体系统（Typography System）

| 字体角色 | 字体名称 | 使用场景 |
|---|---|---|
| **标题字体** | Inter Bold / 思源黑体 Bold | H1-H3 标题 |
| **正文字体** | Inter Regular / 思源宋体 | 正文内容 |
| **代码字体** | JetBrains Mono | 代码块、命令行 |

**字号规范：**
- H1：48px / 3rem
- H2：36px / 2.25rem
- H3：24px / 1.5rem
- 正文：16px / 1rem
- 说明文字：14px / 0.875rem

### 布局系统（Layout System）
- **最大宽度**：1200px（桌面端）
- **内边距**：24px（移动端）/ 48px（桌面端）
- **栅格系统**：12 列栅格，间距 24px
- **圆角**：8px（小组件）/ 16px（卡片）/ 24px（大区块）

### 组件规范（Component Guidelines）

**按钮（Button）：**
```css
/* 主要按钮 */
background: linear-gradient(135deg, #FF3B3B, #7B61FF);
padding: 12px 24px;
border-radius: 8px;
font-weight: 600;

/* 次要按钮 */
border: 1px solid #00D4FF;
color: #00D4FF;
background: transparent;
```

**卡片（Card）：**
```css
background: rgba(255, 255, 255, 0.05);
border: 1px solid rgba(255, 255, 255, 0.1);
border-radius: 16px;
backdrop-filter: blur(10px);
```

## 设计原则
- **深色科技风**：主色调为深蓝黑，营造专业、前沿的技术感。
- **高对比度**：确保文字与背景的对比度符合 WCAG AA 标准（4.5:1 以上）。
- **渐变点缀**：使用红-紫渐变作为点缀，增加视觉层次感，但不过度使用。
- **留白优先**：保持充足的留白，避免视觉拥挤。

## 注意事项与避坑指南
- **禁止随意使用颜色**：所有颜色必须来自上述色彩系统，不得使用规范外的颜色。
- **禁止混用字体**：在同一页面中，标题和正文字体必须统一，不得随意混用。
- **移动端优先**：所有视觉内容必须先确保移动端的可读性，再优化桌面端。
