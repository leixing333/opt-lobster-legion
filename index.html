A2A 两步触发协议
Agent 间通信必须先发送可见锚点消息，再通过 sessions_send 触发。确保所有协作在人类视线范围内，杜绝 AI 团队失控。
锚点消息 sessions_send 可见留痕
🏖️
沙箱物理隔离
每个 Agent 拥有独立的工作区（workspace-{agentId}），不得越界访问其他 Agent 的私有文件，从根本上防止上下文污染。
独立工作区 最小权限 防污染
📊
QAPS 任务分类
所有任务按 Q（问题）、A（行动）、P（项目）、S（系统变更）分类，不同类型有不同的执行规范和验收标准，确保流程标准化。
Q: 一次性问题 A: 有交付物 P: 跨天项目 S: 系统变更

🧠 记忆系统

三层记忆，知识永不丢失

从对话记录到组织资产，经验在三层架构中逐级压缩、提炼，最终成为可跨项目复用的核心竞争力。

L0
原始对话
Slack / 飞书 聊天记录，用于审计和追溯，不直接复用。
1x
原始信息密度
L1
Closeout 摘要
每个任务完成后生成 10-15 行结构化总结，存入 memory/YYYY-MM-DD.md。
~25x
信息压缩比
L2
RAG 知识库
KO 从 L1 中提炼原则、模式和踩坑记录，构建可语义检索的长期知识库。
~100x
信息压缩比

自主等级 (Autonomy Ladder)

L0
只建议，不动手
提供方案选项，等待人类决策。
L1
可逆操作，直接执行
写草稿、做调研、整理文档。
L2
可回滚，执行后汇报
提 PR、改配置、写分析报告。
L3
不可逆，必须人类确认
发布上线、对外发邮件、付款。

15 分钟，组建你的龙虾军团

从零开始，到 6 个 AI 数字员工协同运转，只需三步。

01
📦
克隆并配置
下载系统文件，复制 .env.example 为 .env，填入 API Keys 和 Slack 频道 ID。
02
⚡
一键部署
运行 deploy.sh，脚本自动安装 OpenClaw、部署 6 个 Agent、安装技能，约 5 分钟完成。
03
🦞
唤醒军团
在 Slack #hq 频道发送第一条消息，CoS 幕僚长将自动完成 A2A 闭环配置并汇报就绪。
# 步骤 1：克隆系统
git clone https://github.com/your-name/opt-lobster-legion.git
cd opt-lobster-legion

# 步骤 2：配置环境变量
cp .env.example .env
# 编辑 .env，填入 ANTHROPIC_API_KEY、SLACK_BOT_TOKEN 等

# 步骤 3：一键部署
chmod +x deploy.sh && ./deploy.sh

# 步骤 4：验证部署
openclaw status # 应显示 6 个 Agent 全部 running

# 在新设备恢复（拷贝即升级）
openclaw backup restore --name "opt-2026-03-20"
